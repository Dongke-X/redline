长页面 / 滚动场景 适配演示
v1.0 · 2026-05-09
01 · INTRO

长内容里的反馈，应该和短内容一样自然

这是一个非 deck 风格的长滚动页面。widget 应当能够：

  • 跟随滚动，工具栏与选中元素的相对位置稳定
  • 当前页（section）通过 IntersectionObserver 自动切换，反馈面板里"当前页"标签随之更新
  • 框选标注在页面任意区域都能精准记录其相对 section 的百分比位置
  • 面板（F）与 deck-stage overlay 的浮动位置不被滚动打乱
简单页面是基本功，长页面才暴露功能边界。本页有 7 个 section，每个 section 至少占一屏高度。

03 · EDIT

元素级编辑：移动 / 缩放 / 旋转 / 字体 / 隐藏 / 删除

按 E 进入编辑模式，单击元素选中，双击直接修改文字。这一段可以双击改写——把"这一段可以双击改写"改成你想要的版本。

演示卡片 A
用于测试卡片整体的 transform 操作

选中这张卡片，拖角试试缩放，拖角外（对角线方向）旋转。试试 Aa 字体按钮看是否可以切换字体。

演示卡片 B
用于测试相邻卡片之间的影响

编辑模式下隐藏 / 删除某一张卡片，看其他卡片的 layout 反应。

04 · FEEDBACK

三种反馈形态：全局 / 每页 / 每个元素

全局反馈

对整体感受的留言。按 F 打开面板，最下面那条 textarea。

每页反馈

面板的"当前页"区会随滚动自动切换 section。这条段落归属于"04 · FEEDBACK"。

每个元素反馈

选中元素后，工具栏里有 💬 按钮。点开能写针对该元素的备注，给 agent 看。

05 · MARQUEE

任意位置框选 + 上传图片 / 写文字

点击 deck 底部 overlay 里的框选按钮（FAB 模式下不显示——见 issue），然后在画面任意区域拖动画一个矩形框。

松开后小绿色 pill 工具条出现：

  • ✎ 写文字反馈
  • 📷 上传参考图片
  • ✕ 删除该标注

关键测试：在不同 section 各画一个框，刷新页面，看是否都还在原位（rectPct 是否随 section 缩放正确）。

这里是一个虚拟图片占位区
06 · I18N

中文 ⇄ 英文 一键切换

面板顶部有 🌐 图标按钮，点击在 zh / en 间切换。当前 locale 会持久化到 localStorage，刷新仍然是上次的设置。

默认按 navigator.language 自动检测——浏览器语言以 zh 开头默认中文，否则英文。

所有 toast / 模态 / 占位符 / 按钮 tooltip
80+ keys 双语

切换后所有文字立刻变为目标语言。

07 · CHECK

滚动测试 checklist

  • 滚动到任意位置，按 F 打开面板，看面板浮动位置不变（fixed）
  • 反馈面板"当前页"标签随滚动切换（IntersectionObserver）
  • 选中元素后滚动，工具栏 + 4 角 handles 跟随元素位置
  • 编辑模式下做 5+ 处改动，关闭编辑模式（按 E）→ 看到 toast "编辑模式已关闭 · N 处改动暂存中"
  • 不同 section 各画 1 个框选标注，刷新页面，看标注是否回到原位
  • 切换中英文，所有文字应同步切换
  • 保存反馈 → ~/.feedback-widget/sessions/ 应有 .json + .md + 图片附件

跑完上面 checklist 没问题，就说明长页面适配到位。
